# Event Search in the US <br/>
A web application that search (by keywords) and display information of the events that are happening in the U.S

## Technologies USED <br/>
1. HTML5
2. CSS3
3. JavaScript
4. jQuery
5. Google Fonts
6. Ticketmaster API
